# landing-page-library
Proyecto de portafolio.
Constituido por un archivo principal index.js en tipo modulo para la adicion de los demas documentos JS para el consumo de datos desde el archivo .json, animaciones, responsive y validaciones de formularios.
CSS distribuido en tres archivos principalmente index.css y variables.css en el último se declaran las variables principales que usamos en toda la página, importados con @import a los demás archivos .css, se añaden animaciones básicas como a botones, los libros renderizados dentro de cada sección.
HTML se utiliza un página principal para el inicio y un esqueleto reutilizable para las paginas de cada categoria de los libros dentro de la libreria.
Se hace uso de API Fetch para el consumo de los datos dentro del json.
